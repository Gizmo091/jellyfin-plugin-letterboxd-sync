Watchlist Sync

Paste a watchlist link (letterboxd.com or boxd.it short link) or a Letterboxd username to sync their public watchlist as a playlist. No login required. Tick Auto-request to have films missing from your library requested in Seerr (configure Seerr below).

Seerr Integration

Optional. Set a Seerr / Jellyseerr / Overseerr instance so watchlists marked Auto-request can request missing films. Requests are made on behalf of the matching Jellyfin user, so Seerr applies that user's own approval and quota rules. These settings are server-wide (shared by all users).
Base URL of your Seerr instance, e.g. http://localhost:5055. Leave empty to disable auto-requesting.
